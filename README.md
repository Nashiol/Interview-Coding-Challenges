# Coding Interview Solutions

Welcome to my coding interview solutions repository!

This repository contains my solutions to programming and algorithm challenges from various coding platforms. The goal is to improve my problem-solving skills, strengthen my understanding of data structures and algorithms, and prepare for technical interviews or question.

## Repository Structure

Each problem is organized into its own project folder.

Example:

```text
.
├── Two Sum/
│   ├── solution.py
│   ├── README.md
│   └── notes.md
├── Valid Parentheses/
│   ├── solution.py
│   ├── README.md
│   └── notes.md
└── ...
```

Each problem folder may include:

* **Solution** – The implementation of the problem.
* **README.md** – Where the question context will be.
* **Notes** – Personal notes, observations, or alternative solutions (when applicable).

## Related Project

### LeetLog

As I solve coding challenges, I also use **LeetLog** to track my progress.

LeetLog is a personal platform I built to log solved LeetCode problems, record notes and insights, track revision history, and schedule problems for future review and re-practice. It helps me reinforce concepts through consistent revision instead of solving each problem only once.

The combination of this repository and LeetLog allows me to:

* Keep all my solutions in one place.
* Document different approaches and lessons learned.
* Review previously solved problems.
* Re-practice challenges to strengthen long-term problem-solving skills.
* Track my growth throughout my interview preparation journey.

> **LeetLog** is an ongoing project that supports my coding interview preparation by making revision structured and consistent.
